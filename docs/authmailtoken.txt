Using mailtoken authentication with JANUS (and SimpleSAMLphp)
=============================================================

To use mailtoken as an authentication source you need to do the following.

  * Enable the module (At the moment mailtoken is part of the JANUS module)
  * Set up the database
  * Set up the authsources.php
  * Edit mail text

First thing to do is to enable the module

    touch PATHTOSIMPLESAML/module/janus/enable

After that you need to set up the table, that mailtoken uses. First create
a database e.g. MySQL, PostgreSQL ect.) You can only use dabases that have
drivers implemented in PDO.

Here is the initialization SQL script for MySQL

    CREATE TABLE `janus__tokens` (
      `id` int(11) NOT NULL auto_increment,
      `mail` varchar(320) NOT NULL,
      `token` varchar(255) NOT NULL,
      `notvalidafter` varchar(255) NOT NULL,
      `usedat` varchar(255) default NULL,
      PRIMARY KEY  (`id`),
      UNIQUE KEY `token` (`token`)
    ) ENGINE=MyISAM DEFAULT CHARSET=latin1

After you have set up your database you need to modify authsources.php in
config dir. Change the code below to fit your needs.
    
    'mailtoken' => array(
        'janus:MailToken',
        'dsn' => 'mysql:host=localhost;dbname=database_db',
        'username' => 'THEUSERNAME',
        'password' => 'THEPASSWORD',
        'table' => 'janus__tokens',
    ),

The last thing you need to do is to edit MailToken.php. Open the file and
locate the method newToken. Here you will be able to edit the appearance of
the email send by Mailtoken. REMEMBER to change the from and reply-to
addresses.

If you are using the JANUS module install script, the table will be created
for you and the config for the authsources.php as well.
